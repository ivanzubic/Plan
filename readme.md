README

Naziv projekta:
F1 igra u Pythonu (Pygame)

Opis:
Ovaj projekt je jednostavna 2D F1 trkaća igra izrađena u Pythonu koristeći Pygame biblioteku. Igra omogućava upravljanje bolidom, izbjegavanje prepreka i natjecanje za najbolji rezultat. Projekt je namijenjen učenju osnova game developmenta i rada s Pygame-om.

Instalacija:
1. Instaliraj Python (preporuka: Python 3.10+)
2. Instaliraj Pygame:
   pip install pygame
3. Kloniraj GitHub repozitorij ili preuzmi projekt
4. Otvori projekt u Visual Studio Code (VS Code)

Korištenje (pokretanje igre):
1. Otvori terminal u VS Code-u
2. Pokreni igru naredbom:
   python main.py

Kontrole:
- Strelica lijevo/desno: pomicanje bolida
- Strelica gore: ubrzanje
- Strelica dolje: kočenje

Struktura projekta:
- main.py – glavna datoteka igre
- /assets – slike (bolid, staza, pozadina)
- /sounds – zvukovi (ako postoje)
- /src – dodatni moduli (logika igre)

Zahtjevi:
- Python 3.10+
- Pygame
- VS Code (preporučeno)
